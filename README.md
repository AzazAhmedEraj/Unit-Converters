# Unit Converter

A Python-based unit converter for performing a wide range of conversions across different measurement types, including length, weight, and temperature.

## Features

- **Length Conversions**:
  - Meters ↔ Kilometers
  - Miles ↔ Kilometers
  - Inches ↔ Centimeters
  - Feet ↔ Meters

- **Weight Conversions**:
  - Kilograms ↔ Pounds
  - Grams ↔ Kilograms
  - Ounces ↔ Grams

- **Temperature Conversions**:
  - Celsius ↔ Fahrenheit
  - Celsius ↔ Kelvin
  - Fahrenheit ↔ Kelvin

## Requirements

- Python 3.x

No external libraries are required for this script.
